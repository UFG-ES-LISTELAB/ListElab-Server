# Testes Automatizados com Postman

## Configuração
Abra o Postman e importe os arquivos .json presentes neste diretório, que são dos tipos:

- Coleção, que é onde são definidas as chamadas à API e testes
- Ambiente, onde as variáveis para execução das requisições são definidas (ex.: URL da API)

Em seguida, vá em 'Settings/General' e desative a opção 'SSL certificate version'.

## Executar testes
Para executar os testes é necessário primeiro escolher o ambiente que será testado (ListElab API ou ListElab Localhost). Isto pode ser definido no canto superior direito (Environments). Feito isso, clique no botão de 'Play' na coleção e em seguida clique em 'Run'. Na nova janela que se abrirá, clique em 'Run ListElab'.
